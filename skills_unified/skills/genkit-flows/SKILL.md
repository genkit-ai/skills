# Genkit Flows

Flows are the building blocks of Genkit applications, encapsulating logic, schema validation, and streaming.

## Capabilities

*   **Logic Definition**: Encapsulate code in a trackable flow.
*   **Validation**: Validate inputs and outputs with Zod.
*   **Streaming**: Stream partial results to the client.
